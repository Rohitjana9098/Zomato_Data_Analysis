# Zomato Data Analysis

## Overview
This repository contains a data analysis project focused on Zomato restaurant data. The analysis explores various trends, customer preferences, and factors affecting restaurant ratings and pricing.

## Files in the Repository
- `Zomato_Data_Analysis.ipynb`: The primary Jupyter Notebook containing the data exploration, preprocessing, visualizations, and analytical insights.
- `Zomato data .csv`: The dataset used for the analysis, providing detailed information about various restaurants, cuisines, costs, and customer ratings.

## Usage
To view or run the analysis:
1. Clone this repository locally or download the files.
2. Install the necessary Python packages (such as `pandas`, `matplotlib`, `seaborn`, `jupyter`).
3. Open `Zomato_Data_Analysis.ipynb` in Jupyter Notebook or JupyterLab to explore the code and visualizations.
